
# Ransomware

A Social Media App where everyone can login with there details
and post on app.




## Tech Stack

**Client:** React, Redux, Css

**Server:** Firebase


## Demo

https://ransomware121.netlify.app


## Running Tests

To run tests, run the following command

```bash
  npm install
```
```bash
  npm run dev
```
